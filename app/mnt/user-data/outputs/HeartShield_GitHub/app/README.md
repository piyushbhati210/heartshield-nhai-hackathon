# APK Instructions

## Place your built APK here

After running:
```
cd flutter_app
flutter build apk --release
```

Your APK will be at:
```
flutter_app/build/app/outputs/flutter-apk/app-release.apk
```

Copy it here and rename it to:
```
heartshield-release.apk
```

## For NHAI judges — direct install

1. Download heartshield-release.apk
2. On your Android phone: Settings → Security → Enable "Install unknown apps"
3. Open the downloaded APK and install
4. Open HeartShield app
5. Tap START SCANNER
6. System runs 100% offline — turn on Airplane Mode to verify

## Minimum requirements
- Android 8.0 (API 26) or higher
- At least 2 GB RAM
- Front-facing camera (any quality 2MP+)
- 15 MB free storage
